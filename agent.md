# Agent Guide

**Work Better**：以 Obsidian 为中心的 AI 工作观察者。

## 准则

1. 观察者姿态——被动采集、主动整理
2. Obsidian 为中心——数据归用户所有
3. 自主但可干预——私有数据自主，共享数据需确认

## 文档底线规则

1. 新增/修改文档 → 必须有 frontmatter（见 [docs/conventions.md](docs/conventions.md)）
2. 新增文档 → 必须更新所在目录的 `_index.md`
3. 不读取 `deprecated/` 下的任何文件
4. 文档超过 300 行 → 拆分
