# Work Better - 功能索引

> **维护说明**：当新增、移除或修改功能时更新本文档。
> 本文档作为功能的完整索引，可用于指导测试计划制定。
> 每个功能项标注所属模块和当前状态。

## 状态标记

- `⬜ 未开始` - 代码不存在，或仅有接口定义无实现
- `🟡 开发中` - 代码存在，仅 L1 单元测试
- `🧪 集成验证` - 代码存在，L1 + L2 测试通过（真实后端验证）
- `✅ 已完成` - 代码存在，L1 + L2 + L4/L5 测试通过（端到端可用）

> **说明**：2026-06-11 按照 [功能完成标准](completion-criteria.md) 重新评估了所有功能状态。
