# 命令面板作为主要导航方式

Work Better 采用命令面板（⌘K）作为主要导航方式，而非传统的菜单或侧边栏。这个决策参考了 Linear、Raycast、Arc Browser 等现代生产力工具的设计模式。

## 决策

- 命令面板支持混合模式：导航、操作、搜索
- 所有功能都应可通过命令面板访问
- 快捷键 ⌘K 作为全局触发方式
- 支持自然语言输入和特殊前缀（> 执行命令、# 搜索事件、@ 搜索任务）

## 理由

- 键盘优先是桌面应用的核心优势
- 降低功能发现成本（用户不需要记住菜单位置）
- 支持未来 AI 集成（自然语言命令）
- 符合现代生产力工具的设计趋势
